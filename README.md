## Installation

```bash
$ npm install
```

## Running the app

Copy the .env.example file to .env and fill in the values.

Run prisma migrations

```bash
$ npm run migration-dev
```

Run app locally

```bash
# development
$ npm start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
